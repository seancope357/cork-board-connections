# Custom Hooks

This directory contains reusable React hooks for the application.

## Planned hooks:
- `useBoard.ts` - Board state management
- `useConnections.ts` - Connection management logic
- `useDragAndDrop.ts` - Drag and drop functionality
- `useResize.ts` - Item resizing logic
- `useAutoSave.ts` - Auto-save functionality
- `useKeyboardShortcuts.ts` - Keyboard shortcuts handler
- `useUndoRedo.ts` - Undo/redo state management
